# Night School System

Lisa Night School is the designated learning and self-improvement loop for Lisa.

## Flow:

```txt
collect daily traces
→ mine mistakes
→ run brain debate
→ quiz Planner / Feasibility / Ranker
→ score each brain
→ create improvement candidates
→ run evals
→ Auditor creates report cards
→ Auditor creates marksheet
→ update Agent Memo
→ send morning report to Telegram
```

Night School must not silently activate improvements.

All activation requires Policy OS and Telegram validation.
