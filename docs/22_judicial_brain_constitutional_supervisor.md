# Judicial Police / Constitutional Supervisor

The Judicial Police (`JudicialBrain`) is the top-level constitutional supervisor in Lisa's architecture.

## Role
It inspects all major lifecycle actions: Autopilot activation, brain swarm outputs, token reservations, tool adoption, and sandbox executions.

## Authority
The Judicial Police can return a `ConstitutionalVerdict` to allow, warn, require approval, pause, stop, or block an action. It cannot be bypassed, disabled, or overridden by any other component.
