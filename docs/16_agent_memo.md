# Agent Memo System

Agent Memo is Lisa’s persistent academic record.

It tracks:

```txt
Night School sessions
brain report cards
marksheets
learning curves
teacher effectiveness
skill replacements
skill retirements
capability recommendations
MCP/package recommendations
approved capabilities
rejected capabilities
Telegram approvals
rollback paths
```

Agent Memo must be append-only by default.

Skill replacement records must preserve:

```txt
old skill
new candidate skill
reason for replacement
teacher lesson
auditor score
eval result
risk level
approval id
rollback path
activation date
```
