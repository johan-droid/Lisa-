# Capability Allocation System

Auditor can recommend packages/MCPs/tools/skills for specific brains, but cannot grant them directly.

## Flow:

```txt
Auditor detects capability gap
→ recommends package/MCP/skill
→ Cyber-Immune scans it
→ Policy OS evaluates it
→ DevShell sandbox tests it
→ Teacher creates training/eval
→ Auditor verifies readiness
→ Telegram validation required
→ capability lease created if approved
```

Capability access must be lease-based:

```txt
task-scoped
brain-scoped
time-limited
revocable
audited
reported to Telegram
```
