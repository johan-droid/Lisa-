# General User Job Router

Lisa handles a variety of job types through the `JobRouter` component.

## Supported Jobs
- planning_job: Architecture and plans.
- research_job: Online tool finding (Teacher led).
- implementation_job: Code changes via Sandbox.
- tool_adoption_job: Discovering and promoting external tools.
- maintenance, security, autopilot, spike, and unknown jobs.

The selected job type influences the loaded Constitution and dictates the operational flow.
