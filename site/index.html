7-brain architecture

Every brain has a bounded responsibility.

No brain is allowed to silently bypass Judicial Police, sandbox boundaries, audit logging, or Principal approval gates.

Planner

Breaks the user goal into ordered steps, assumptions, risks, outputs, and approval checkpoints.

Feasibility

Checks missing files, permissions, dependencies, infrastructure limits, and implementation blockers.

Ranker

Scores clarity, safety, feasibility, rollback safety, token efficiency, and readiness.

Teacher

Discovers tools, packages, SDKs, CLIs, MCPs, and integration options.

Cyber-Immune

Reviews supply-chain risk, permissions, install behavior, and suspicious runtime activity.

Red Team Mirror

Adversarially reviews plans, commands, tool choices, model routing, and escalation paths.

Principal / Auditor

Approves risky evolution, memory promotion, tool trust, merge, deploy, and policy changes.

Trust system

Autopilot is powerful, not lawless.

Lisa can move fast only inside the active Constitution. Judicial Police can block, pause, stop, quarantine, or escalate to Principal approval before risky work continues.

Always blocked

  • Disabling Judicial Police, AuditLog, or SandboxRuntime
  • Bypassing approval gates or sandbox boundaries
  • Credential exfiltration or trusted-host secret access
  • Replacing the public site with generated status output

Approval required

  • Tool promotion from quarantine to trusted use
  • Package, MCP, CLI, or Docker activation
  • Merge to main, deploy, env changes, or external messages
  • Any future edit to the site/ public landing page
