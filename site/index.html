Trust system

Autopilot is powerful, not lawless.

Lisa can move fast only inside the active Constitution. Judicial Police can block, pause, stop, quarantine, or escalate to Principal approval before anything risky continues.

Always blocked

  • Disabling Judicial Police, AuditLog, or SandboxRuntime
  • Bypassing approval gates or sandbox boundaries
  • Destructive shell commands such as rm -rf, curl | sh, chmod 777
  • Secret access, deployment, or database writes without approval

Approval required

  • Tool promotion from quarantine to trusted use
  • Package, MCP, CLI, or Docker activation
  • Large token-bank reservations
  • Merge to main, deploy, env changes, or external messages
